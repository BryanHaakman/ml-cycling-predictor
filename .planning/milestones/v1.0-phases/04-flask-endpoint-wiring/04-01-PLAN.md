```python
# webapp/auth.py
"""Shared authentication decorators for Flask routes and Blueprints."""
import logging
from functools import wraps
from typing import Callable

from flask import jsonify, request

log = logging.getLogger(__name__)


def _require_localhost(f: Callable) -> Callable:
  """Restrict a route to localhost-only access.

  Returns HTTP 403 for any request not originating from 127.0.0.1 or ::1.
  Applied to all /api/pinnacle/* routes and existing /admin routes.
  """
  @wraps(f)
  def decorated(*args, **kwargs):
    if request.remote_addr not in ("127.0.0.1", "::1"):
      return jsonify({"error": "Admin access is restricted to localhost"}), 403
    return f(*args, **kwargs)
  return decorated
```

**`pinnacle_load()` implementation steps (in order):**
1. Call `fetch_cycling_h2h_markets()` — wrapped in try/except ladder (see below)
2. Group results into `dict[str, list[OddsMarket]]` by `market.race_name` (per D-04)
3. Instantiate `NameResolver()` once (outside all loops)
4. For each race group: call `fetch_stage_context(race_name)` once
5. For each `OddsMarket` in the race group: call `resolver.resolve(rider_a_name)` and `resolver.resolve(rider_b_name)`
6. Assemble pair dict using ALL fields from the frozen schema (D-03 hint fields always present — null when not applicable)
7. Assemble race dict: `race_name`, `stage_resolved` (from `stage_context.is_resolved`), `stage_context` (via `dataclasses.asdict(stage_context)`), `pairs`
8. Return `jsonify({"races": races})`, HTTP 200

| localhost → Flask API | _require_localhost gate; only 127.0.0.1/::1 reach route logic |
| Flask → Pinnacle external API | Session cookie sent outbound; Pinnacle response parsed and forwarded |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Spoofing | _require_localhost | mitigate | Check request.remote_addr directly (not X-Forwarded-For); localhost-only gate enforced in webapp/auth.py on both routes |
| T-04-02 | Information Disclosure | PINNACLE_SESSION_COOKIE | mitigate | Cookie read from env var only — never logged, never included in API response or error body; audit log in data/odds.py masks the token |
