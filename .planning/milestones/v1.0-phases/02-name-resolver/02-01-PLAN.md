    """Result of a name resolution attempt."""
    url: Optional[str]

    best_score: Optional[int]

**NameResolver class:**

`__init__(self)`:
- Call `get_db()` and execute `SELECT url, name FROM riders` to load all riders (per D-01, D-02)
- Store as `self._corpus: list[tuple[str, str]]` — list of (url, name) tuples
- Build `self._name_to_url: dict[str, str]` — mapping original name -> url for exact match
- Build `self._normalized_index: dict[str, str]` — mapping `_normalize_pcs_name(name)` -> url for normalized match (per Pitfall 2 in RESEARCH: do NOT use _normalize_name with reversal for PCS names)
- Build `self._corpus_normalized: list[str]` — parallel list of `_normalize_pcs_name(name)` for fuzzy matching (Plan 02)
- Load persistent cache via `self._cache = _load_cache()`
- Log rider count: `log.info("NameResolver: loaded %d riders from cache.db", len(self._corpus))`

`resolve(self, pinnacle_name: str) -> ResolveResult`:
- Stage 1 (cache): if `pinnacle_name in self._cache`, return `ResolveResult(url=self._cache[pinnacle_name], best_candidate_url=None, best_candidate_name=None, best_score=None, method="cache")`
- Stage 2 (exact): if `pinnacle_name in self._name_to_url`, return `ResolveResult(url=self._name_to_url[pinnacle_name], ..., method="exact")`
- Stage 3 (normalized): compute `normalized = _normalize_name(pinnacle_name)`. If `normalized in self._normalized_index`, get url, call `self.accept(pinnacle_name, url)` to promote to cache, return `ResolveResult(url=url, ..., method="normalized")`
- Stage 4 (placeholder for Plan 02): return `ResolveResult(url=None, best_candidate_url=None, best_candidate_name=None, best_score=None, method="unresolved")`

`accept(self, pinnacle_name: str, pcs_url: str) -> None` (per D-08):
- `self._cache[pinnacle_name] = pcs_url`
- Call `self._save_cache()`

**Private helpers:**

`_normalize_name(name: str) -> str`:
- NFKD decomposition: `unicodedata.normalize("NFKD", name)`
- ASCII encode+decode: `.encode("ascii", "ignore").decode("ascii")`
- Lowercase + strip
- Last-word-to-front reversal: `f"{tokens[-1]} {' '.join(tokens[:-1])}"` if >= 2 tokens
- This handles "ROGLIC PRIMOZ" -> "primoz roglic", "VAN AERT WOUT" -> "wout van aert"

`_normalize_pcs_name(name: str) -> str`:
- Same as _normalize_name but WITHOUT the word-order reversal
- "Primož Roglič" -> "primoz roglic", "Wout van Aert" -> "wout van aert"
- CRITICAL: This is a separate function, NOT the same as _normalize_name (per Pitfall 2 in RESEARCH)

`_load_cache(self) -> dict[str, str]`:
- Try to open CACHE_PATH and json.load()
- Handle FileNotFoundError: return {}
- Handle json.JSONDecodeError: log.warning, return {}
- Validate each value against CACHE_URL_PATTERN (per D-07)
- Log and skip invalid entries: `log.warning("_load_cache: skipping invalid entry %r -> %r", key, val)`
- Return validated dict

`_save_cache(self) -> None`:
- Use tempfile.NamedTemporaryFile(mode="w", dir=dirname, suffix=".tmp", delete=False)
- json.dump(self._cache, f, indent=2, ensure_ascii=False)
- os.replace(tmp_path, CACHE_PATH)
- Wrap in try/except OSError, log.warning on failure
