## Odds Format

Prices are returned as **American integer odds** (e.g., `-154`, `+107`).

Conversion to decimal:
- American > 0: `decimal = american / 100.0 + 1.0`
- American < 0: `decimal = 100.0 / abs(american) + 1.0`

Example: `-154` → `1.6494`, `+107` → `2.07`

## Key Extraction via Playwright

The X-Api-Key is a rotating guest token extracted from Pinnacle's website JavaScript. When the key expires, use Playwright MCP browser tools to navigate to pinnacle.com, open DevTools Network tab, filter for requests to guest.api.arcadia.pinnacle.com, and copy the X-Api-Key header value from any request. Store the new value in `PINNACLE_SESSION_COOKIE`.
