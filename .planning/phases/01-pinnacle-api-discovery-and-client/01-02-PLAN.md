**Private helper — audit log (per D-09, D-10, ODDS-02):**
```python
def _append_audit_log(
    markets: list[OddsMarket],
    fetch_status: str,
    error: Optional[str] = None,
) -> None:
    """Append one JSONL record to data/odds_log.jsonl.

    Called after every fetch — including empty fetches and auth errors (per D-10).
    Records post-normalization decimal odds only (per D-09).

    Args:
        markets: List of OddsMarket returned by this fetch (may be empty).
        fetch_status: One of "ok", "empty", "auth_error", "error".
        error: Optional error message string for non-ok statuses.
    """
    record = {
        "fetched_at": datetime.utcnow().isoformat() + "Z",
        "status": fetch_status,
        "market_count": len(markets),
        "markets": [dataclasses.asdict(m) for m in markets],
    }
    if error:
        record["error"] = error
    with open(ODDS_LOG_PATH, "a", encoding="utf-8") as f:
        f.write(json.dumps(record) + "\n")
```

1. Call _get_api_key() — raises PinnacleAuthError if env var absent
2. Build headers: `{"X-Api-Key": api_key, "Referer": "https://www.pinnacle.com/", "Accept": "application/json"}`
3. GET `{PINNACLE_API_BASE}/sports/{PINNACLE_CYCLING_SPORT_ID}/leagues?all=false` with `timeout=REQUEST_TIMEOUT`
4. Call `_check_auth(leagues_resp)` — raises PinnacleAuthError on 401/403
5. Call `leagues_resp.raise_for_status()` for other non-2xx errors
6. For each league in the leagues list:
   a. GET `{PINNACLE_API_BASE}/leagues/{lid}/matchups` with headers and timeout
   b. If response body is not a list (e.g., auth error envelope dict), log warning and `continue` — do not raise
   c. GET `{PINNACLE_API_BASE}/leagues/{lid}/markets/straight` with headers and timeout
   d. If response body is not a list, log warning and `continue`
   e. Build `market_by_id = {m["matchupId"]: m for m in markets_list}`
   f. For each matchup: look up matching market; skip if not found or `market["status"] != "open"`
   g. Extract `prices = {p["designation"]: p["price"] for p in market["prices"]}`
   h. Build OddsMarket with `_american_to_decimal(prices["home"])` and `_american_to_decimal(prices["away"])`
   i. Cast `matchup["id"]` to `str` for `matchup_id`
7. Determine `fetch_status`: `"ok"` if len(markets) > 0 else `"empty"`
8. Call `_append_audit_log(markets, fetch_status)` — always, before return
9. Return `markets` (list, possibly empty — never raise for empty per D-11)
